# Advanced Corporate Analytics Consultant | PlantLuv System Launch

This repository contains the end-to-end operational architecture, financial sensitivity models, and data artifacts built to validate a $1.1M market entry framework for a consumer product launch.

## Repository Contents
*   **`Team2_SU-25-Fin-Excel.xlsx`**: The core quantitative financial engine. Contains the 10-year multi-scenario capital budgeting models, CAPM-based discount rate regressions tracking gross margin volatility against the S&P 500, and supplier EOQ/ROP inventory policy optimization configurations.
*   **`Team2_SU-25-FULLCASE.pdf`**: The final 55-page strategic evaluation. Includes the comprehensive cross-functional analysis linking operational capacity management to target GTM (Go-To-Market) segmentation data.
*   **`Team2_SU-25-CPA.png`**: The 19-node project management dependency network mapping out PERT expected durations and the strategic execution of critical path network crashing.
